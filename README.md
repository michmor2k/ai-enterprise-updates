# AI Enterprise Updates

A self-updating dashboard of the day's top enterprise-AI news — model & platform
releases, agents, orchestration, AI support, philosophy, and strategy — curated
from 13 industry sources, with each article flagged as independent reporting or
vendor-leaning. Save articles (stored privately in your own browser) and browse
them grouped by month.

**Live site:** see the GitHub Pages URL in this repo's About section.

Data refreshes automatically each morning. Built with a static HTML dashboard
that loads its article data from `articles.js`.
